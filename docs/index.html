Full-Featured

Home shelves, radio, synced lyrics, mixtapes, skins, mini player, video — and a plugin ecosystem for everything else. All the features of a modern streaming app, working for your own library.

High-Quality Sound

Gapless playback, crossfade, 10-band EQ, and ReplayGain across FLAC, ALAC, OPUS, and more. An optional native mpv engine adds exclusive audio output for serious listening.

Optimized to the Core

Sub-second startup, instant search, under 200ms playback latency — in a 15MB binary using under 50MB of RAM. Powered by Rust and Tauri — no Chromium, no bloat.

Open Source & Private

100% free and open source. All data stays on your machine. No cloud, no accounts, no telemetry.
