Server directory

A community list of publicly-shared Subsonic and Navidrome servers. Found one you like? Add to Viboplr opens the app with the connection pre-filled — no typing.

These are servers people chose to share publicly. Listings carry shared logins in plain text, pre-filled into the app when you add one. Treat them as public — never list a private server, and don't reuse a password you use elsewhere.

Share your server

Run a Subsonic or Navidrome server you're happy to open up? List it in a minute — the directory lives at servers.viboplr.com.

1

Sign in

Log in with GitHub or Google — no password to create. You manage only the listings you add.

2

Add your server

Give it a name, the server URL, a short description, tags, and the shared login you're comfortable making public.

3

It shows up here

Your listing appears in the directory with a one-click Add to Viboplr button for everyone.

You can edit or remove your listings anytime from your dashboard.
