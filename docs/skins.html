Skins

Theme the entire app — backgrounds, accents, the works. Install any skin from inside Viboplr — Extensions → Skins.

How to add a skin

No pull request, no JSON editing — a bot validates the colors and opens it for review.

1

Export your theme

In the app, open Extensions → Skins → ＋ New skin, tweak the colors in your editor, then hit Submit to gallery — or hand-write a JSON with all 18 color keys.

2

Paste the JSON

Open the submission form and paste the skin JSON (or link to a raw .json URL). Name, author, and type come from it.

3

A bot validates & opens a PR

It checks all 18 colors and your customCSS, then opens a PR that adds your skin file. A maintainer merges it to publish.

Full details in the submission guide.

Made a skin?

Create a skin in the app (Extensions → Skins → ＋ New skin), then submit it with a single form. A bot validates the colors and opens it for review.
